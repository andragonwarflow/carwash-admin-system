
# User Manual - Carwash Admin v2

## 1. Acceso al Sistema
1. Ingrese a la URL del sistema.
2. Introduzca su usuario y contraseña en la pantalla de Login.

## 2. Gestión de Bahías
- **Vista de Bahías**: En el tablero principal, verá tarjetas que representan las bahías.
    - **Verde**: Bahía disponible.
    - **Rojo**: Bahía ocupada.
- **Finalizar Servicio**: Haga clic en el botón "Finalizar" de una bahía ocupada para cerrar la venta, liberar la bahía y marcar el vehículo como completado.

## 3. Gestión de Cola
- **Añadir Vehículo**: Haga clic en "Añadir a Cola" e ingrese el ID del cliente y el servicio solicitado.
- **Procesar Siguiente**: Haga clic en "Procesar Siguiente" para asignar el primer vehículo de la cola a la primera bahía disponible.

## 4. Administración
El sistema gestiona automáticamente:
- El stock de productos al finalizar una venta.
- Los puntos de lealtad de los clientes.
- El registro contable de ingresos.
